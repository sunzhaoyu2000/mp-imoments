The codes try to expand study of finite-N quantum fisher information with respect to nonlocal operators into large-N limit.

The algorithm is from Ref. [1].

The spin-1 models and nonlocal operators are from Ref. [2].

Please download the .zip file and unzip it into a directory "mp-imoments".

Read the driver file " driver_____mp_imoments_v3_SpinOne_StringOrderParameter.m " for more details.
